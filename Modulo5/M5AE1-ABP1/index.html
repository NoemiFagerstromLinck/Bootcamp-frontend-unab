Sistema de Gestión de Pacientes

Información del Sistema

Este sistema permite gestionar los datos de pacientes de un consultorio médico.

Funcionalidades implementadas:

  • Almacenamiento de datos de pacientes
  • Búsqueda de pacientes por nombre
  • Visualización de todos los pacientes
  • Protección de datos con getters y setters
  • Implementado en JavaScript ES5

Para ver el sistema en funcionamiento:

  1. Abre las Herramientas de Desarrollador (F12)
  2. Ve a la pestaña Console
  3. El sistema se ejecutará automáticamente al cargar la página
  4. Podrás ver todos los pacientes y realizar búsquedas

Comandos disponibles en la consola:

  • consultorio.mostrarTodos() - Muestra todos los pacientes
  • consultorio.buscarPorNombre("Juan") - Busca pacientes por nombre
  • paciente1.getNombre() - Obtiene el nombre de un paciente
  • paciente1.setNombre("Nuevo Nombre") - Modifica el nombre de un paciente
